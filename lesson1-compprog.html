Introduction to an Programming

Programming has a rich history that spans a number of decades.

History of Programming Languages presents information pertinent to the technical aspects of the language design and creation. This lesson provides an understanding of the processes of language design as related to the environment in which languages are developed and the knowledge base available to the originators.

The history of programming languages spans from documentation of early mechanical computers to modern tools for software development. Early programming languages were highly specialized, relying on mathematical notation and similarly obscure syntax. Throughout the 20th century, research in compiler theory led to the creation of high-level programming languages, which use a more accessible syntax to communicate instructions.

More Information

The first high-level programming language was Plankalkül, created by Konrad Zuse between 1942 and 1945.The first high-level language to have an associated compiler was created by Corrado Böhm in 1951, for his PhD thesis.The first commercially available language was FORTRAN (FORmula TRANslation), developed in 1956 (first manual appeared in 1956, but first developed in 1954) by a team led by John Backus at IBM.

Ada Lovelace and Charles Babbage collaborated on the Analytical Engine, a rudimentary mechanical computer, in 1883 to create the first programming language. Lovelace recognized the significance of numbers, understanding that they might express more than just numerical quantities. To compute Bernoulli numbers, Lovelace devised an algorithm for the Analytical Engine, the first computer software.

- Ada and the World's First Computer In 1949, assembly language was introduced as a sort of computer programming language capable of simplifying machine code language, which is required to instruct a computer what to do.

- Alick Glennie created Autocode in 1952, which some see as the first compiled computer programming language. This means that it might be converted straight into machine language.

- FORTRAN, a computer programming language for scientific, mathematical, and statistical purposes, was designed by John Backus in 1957.
